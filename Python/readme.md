# Python Scripts

## Libation Helper Scripts

Scripts  that help work with large amount of books from the Libation downloader

## Spotify Playlist Downloader

Downloads and makes a list of songs in your playlists from Spotify, so you  can use them elsewhere.

## Youtube Playlist Maker

Proceses text file lists of songs to make playlists.  HINT: It's useful with  the script just above.

## Blogger Converter

Converts a Blogger blog export of entries to Markdown files.

## Fourchan Downloader

##  Downloads images from a 4chan thread and monitors for new ones (I think)

## GOG ID Extractor

Extracts ids of games from your GOG library.  Useful with some GOG downloader apps to allow you to download them in more manageable chunks.

## Markdown Combiner

Combine many markdown files from a folder into one file.

## Tumblr to Markdown

Converts a Tumbler Export to markdown files.

## Nonsense Scripts

Useless code created just screwing round.  Like incrediblyy inefficient  Hello World scripts.

## URL Extractor

Extracts all URL links in a webpage

## Images URL Extractor

Extracts image URLs from a webpage.
