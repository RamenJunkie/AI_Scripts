# Shell Scripts

## Folder Merge

I ran a script to download all of the snapshots of my website over the years from Archive.org's wayback machine, but it saved them with each snapshot in individual folders and I wanted to merge them all into one complete snapshot.

## Process Zines

Ocasionally I find online zines or books that don't have a PDF download.  If you take snapshots of every page, you often end up with browser cruft around the edges.  With adjustments to the dimensions in the file, this should process all of the files by cropping them down to just the pages, then splitting any wide images into a left and right individual page image.  The result can easily be processed to a single PDF file of single pages.

## Wordpress Image Optimizer

I tried several image optimization plug ins but they all were limited by paid plans  This script can be run on a wordpress image folder (or any image folder really) and it will scale down any image larger than a particular size in order to save file space.
